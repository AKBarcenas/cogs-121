What is MapEats?

MapEats is a cloud based chat bot that unifies Yelp, Google, Wit.AI, and some secret sauce to help you find the best local food joints catered to your preferences!

How do I use MapEats?

Just type what you're craving and Karan (our handy dandy chat bot) will find a restaurant for you!
If you think the result is then Karan can get you directions.
